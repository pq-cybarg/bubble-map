What this layer is for

The formal core (the SCC/Z3/TLA+ proofs on data/graph.json) deliberately excludes relationship/association edges. This overlay collects the graded "who-overlaps-whom" observations — the structural questions worth holding in view — without letting them contaminate the proofs. It is a map of proximity, explicitly not a map of conspiracy.
