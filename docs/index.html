What this is — and isn't

It is a cited inventory of how the AI / banking / credit / commodity / energy / defense / crypto / identity webs connect, with the provable core machine-verified and the rest evidence-graded. It is not a claim of a single conspiracy: the honest reading is a small recurring operator-network + recurring structures (LTCM interconnection, Enron off-balance-sheet vehicles, dotcom vendor financing) + regulatory arbitrage, rebuilt each era in the least-regulated venue. Intent is never inferred from adjacency; sensitive threads (people, influence, foreign actors) are graded and kept out of the proofs.

Reproduce everything: clone the repo and run bash run_all.sh (Python + Z3; Java for TLA+/Alloy — jars auto-fetched). Fork it. Extend it. Make the seeing and the proving cheap for everyone.
